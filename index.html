Database

Java project for managing a mongo database that supports the loading of external jars.

Download .zip Download .tar.gz View on GitHub

NOTE: This is not an implementation.

This project was initially created for Altitude MapDev.

The project strives to be as extendale and configurable as possible.

Issues and pull requests are welcome


Modules

External jars, or "modules" as the project calls them, are jars that are loaded at runtime that add funtionality to the project.

An example module can be found in the "example-module" branch
